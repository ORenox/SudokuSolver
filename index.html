Sudoku Calculator

Introduce number from 1 to 9 in the cells, let empty the cells you want to calculate. Use Resolve to calculate the solution

Hecho con HTML, CSS y JS

Description

This solver implements a backtracking algorithm to find the solution to a Sudoku puzzle. It tries numbers 1-9 in each empty cell and uses constraint propagation to eliminate impossible candidates.

Shortcuts and Tips

  • Only number from 1 to 9 are alowed in the cells
  • Use the "Load Example" button to load a sample Sudoku puzzle
  • Click "Solve" to find a solution for the current puzzle
